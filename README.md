##  Requirements
- **PHP:** 7.4 or higher  
- **MySQL:** 5.7 or higher  
- **Server:** Apache (via XAMPP / WAMP / LAMP)  

##  Installation Guide
1. **Clone or download** this repository.  
2. Copy the project folder to your **web server directory** (e.g., `htdocs/` for XAMPP).  
3. **Import** the `database.sql` file into **phpMyAdmin**.  
4. **Create** a new file named `db_connection.php` based on `db_connection.example.php`, and add your **local database credentials**.  
5. **Run** the project from your browser:  
http://localhost/hospital-management-system/
